# Предобработка HTML-статей

## Stage 1: Очистка Avito-специфичной разметки (BeautifulSoup)

**Скрипт:** `src/preprocess_html.py`

Последовательная обработка DOM в одном soup:

1. `<img>` → alt-текст (или удаление)
2. `<br>` → пробел
3. `<input>` → удаление (переключатели табов/спойлеров)
4. Пустые `<label>` → удаление (иконки спойлеров)
5. `<a>` → только текст (href отбрасывается)
6. `<headline>`, `<chunk>` → unwrap (снятие обёртки)
7. Пустые `<p>`, `<div>`, `<span>`, `<strong>`, `<em>` → удаление
8. `div.factoid_*` → unwrap
9. `div.spoiler` → заголовок + контент (без обёртки)
10. `div.tabset` → заголовки табов + контент каждой панели
11. Вложенные `<table>` → flatten (текст внутренней таблицы встраивается в ячейку)
12. Пустые `<table>` → удаление

Результат: HTML только со стандартными тегами (`p`, `ul`, `ol`, `table`, `strong`, `h1`–`h3` и т.д.).

## Stage 2: Конвертация в текст (html2text)

**Скрипт:** `src/html2text_stage.py`

Настройки:
- `body_width=0` — без переносов строк
- `ignore_emphasis=True` — убрать `**`/`*`
- `ignore_links=True` — убрать `[text](url)`
- `ignore_images=True` — убрать `![alt](src)`
- `single_line_break=True` — схлопывание пустых строк
- `pad_tables=True` — форматирование таблиц
- `unicode_snob=True` — юникод вместо ASCII

После html2text: дополнительное схлопывание пробелов и пустых строк (regex).
